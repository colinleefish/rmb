URI Addressing Scheme
8 scopes
rmb://sessions/<uuid>
Session container -- conversation abstract
rmb://turns/<uuid>
Raw user+assistant exchange (session_id in meta)
rmb://atoms/<uuid>
Typed fact extracted from a session (session_id in meta)
rmb://scenes/<uuid>
Coherent segment with abstract + body facets
rmb://profile
Singleton -- stable identity facts about the user
rmb://preferences/<slug>
Recurring preferences including AI-behavior rules
rmb://entities/<slug>
People, projects, companies, tools
rmb://events/<date-slug>
Dated decisions and milestones -- immutable, append-only
